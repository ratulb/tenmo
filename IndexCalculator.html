struct IndexCalculator

Utility for calculating flat indices and coordinates in multi-dimensional arrays.

Provides static methods for converting between flat (linear) indices and multi-dimensional coordinates, which is essential for tensor operations.

Implemented traits

AnyType, Copyable, ImplicitlyCopyable, ImplicitlyDestructible, Movable, RegisterPassable

Methods

flatten_index

static flatten_index(shape: Shape, indices: IntArray, strides: Strides, offset: Int = 0) -> Int

Calculate flat (linear) index from multi-dimensional indices.

Panic if indices or strides have wrong rank, or if any index is out of bounds.

Args:

  • ​shape (Shape): Shape of the tensor
  • ​indices (IntArray): IntArray of indices, one per dimension
  • ​strides (Strides): Strides of the tensor
  • ​offset (Int): Base offset to add (default: 0)

Returns:

Int: The flat memory offset corresponding to the given indices

index_to_coord

static index_to_coord(shape: Shape, flat_index: Int) -> IntArray

Convert flat (linear) index to multi-dimensional coordinates.

Panic if flat_index is out of bounds.

Args:

  • ​shape (Shape): Shape of the tensor
  • ​flat_index (Int): Linear index to convert

Returns:

IntArray: IntArray of coordinates corresponding to the flat index

max_index

static max_index(shape: Shape, strides: Strides, offset: Int) -> Int

Calculate the maximum valid flat index for the given shape and strides.

Args:

  • ​shape (Shape): Shape of the tensor
  • ​strides (Strides): Strides of the tensor
  • ​offset (Int): Base offset to add

Returns:

Int: The maximum flat index that can be accessed in the tensor.
