__init__

__init__(out self, in_features: Int, out_features: Int, init_seed: Optional[Int] = None, init_method: String = "standard", bias_zero: Bool = True, weight_factor: Scalar[dtype] = 1, profile_samples: Int = 10)

Initialize LinearBLAS layer with configurable weight initialization.

Args:

  • ​in_features (Int): Number of input features.
  • ​out_features (Int): Number of output features.
  • ​init_seed (Optional): Random seed for reproducibility.
  • ​init_method (String): Weight initialization method:
    • "standard": Uniform[-0.1, 0.1] (good for [0,1] normalized inputs).
    • "xavier": Xavier/Glorot uniform (good for tanh/sigmoid).
    • "he": He/Kaiming normal (good for ReLU with standardized inputs).
  • ​bias_zero (Bool): If True, initialize bias to zeros.
  • ​weight_factor (Scalar): Scaling factor for weight initialization.
  • ​profile_samples (Int): Samples per method for profiling:
    • 0: Skip profiling, always use native
    • 1-3: Fast profiling (may be noisy)
    • 5-10: Recommended (default: 10)

    • 10: Extra stable, slower startup.
