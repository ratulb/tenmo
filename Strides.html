struct Strides

Strides for tensor indexing.

Represents the stride values for each dimension of a tensor, indicating how many elements to skip in memory to move to the next element along each dimension.

For a tensor with shape (d0, d1, ..., dn), the strides (s0, s1, ..., sn) define how many elements to traverse in memory to increment the index in each dimension. In C-contiguous layout, strides are calculated as (d1d2...dn, d2...*dn, ..., dn, 1).
