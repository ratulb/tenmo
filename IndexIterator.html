struct IndexIterator[shape_origin: ImmutOrigin, strides_origin: ImmutOrigin]

Iterator over memory offsets for a tensor with given shape and strides.

This iterator generates the physical memory offsets for each logical element in a multi-dimensional tensor. It handles both contiguous and strided tensors efficiently, using an odometer-like increment strategy for non-contiguous data.
