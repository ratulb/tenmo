max_index

max_index(self) -> Int

Calculate the highest accessible memory offset.

For dimensions with positive strides, the maximum is reached at the last index of that dimension. For negative strides, the highest address is already at index 0 (the base offset), so those dimensions do not contribute.

Example:

var buf = NDBuffer[DType.float32](Shape(3, 2), strides=Strides(4, -1), offset=10)
print(buf.max_index())  # 10 + 1*4 = 14

Returns:

Int: The highest valid memory offset.
