struct Shape

Represents the shape (dimensions) of a tensor.

A Shape instance stores the dimensions of a multi-dimensional array, where each dimension represents the size along a particular axis. For example, a 2x3 matrix has shape (2, 3).

Shape instances are immutable - operations that modify dimensions return new Shape instances rather than modifying in place.
