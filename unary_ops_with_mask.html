unary_ops_with_mask[op_code: Int, dtype: DType, simd_width: Int = simd_width_of[dtype](), simd_vectors_per_thread: Int = (2 * simd_width)](result: UnsafePointer[Scalar[dtype], MutAnyOrigin], mask: UnsafePointer[Scalar[dtype], MutAnyOrigin], A: UnsafePointer[Scalar[dtype], ImmutAnyOrigin], size: Int)

Single-pass kernel: compute activation output AND gradient mask.

Both result and mask are written in one GPU pass — no second kernel needed.

For RELU_FORWARD: result[i] = max(A[i], 0) mask[i] = 1.0 if A[i] > 0 else 0.0

Args:

  • ​result (UnsafePointer): Output buffer for activated values.
  • ​mask (UnsafePointer): Output buffer for gradient mask.
  • ​A (UnsafePointer): Input buffer (contiguous, same device).
  • ​size (Int): Total number of elements.
