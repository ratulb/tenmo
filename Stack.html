forward

static forward[track_grad: Bool = True](tensors: List[Tensor[dtype]], axis: Int = 0, requires_grad: Optional[Bool] = None) -> Tensor[dtype]

Stack tensors along a new axis.

Example: A: (2, 3), B: (2, 3) stack([A, B], axis=0) → (2, 2, 3) stack([A, B], axis=1) → (2, 2, 3) stack([A, B], axis=2) → (2, 3, 2).

Args:

  • ​tensors (List): List of tensors to stack (must have identical shapes).
  • ​axis (Int): Position where new dimension is inserted.
  • ​requires_grad (Optional): Whether to track gradients.

Returns:

Tensor: Stacked tensor with new dimension at position 'axis'.

vstack

static vstack[track_grad: Bool = True](tensors: List[Tensor[dtype]], requires_grad: Optional[Bool] = None) -> Tensor[dtype]

Stack tensors vertically (row-wise).

For 2D+ tensors: equivalent to concat(axis=0). For 1D tensors: reshapes to 2D then stacks.

Example: A: (3,), B: (3,) vstack([A, B]) → (2, 3)

A: (2, 3), B: (2, 3) vstack([A, B]) → (4, 3).

Args:

  • ​tensors (List): List of tensors to stack vertically.
  • ​requires_grad (Optional): Whether to track gradients.

Returns:

Tensor: Vertically stacked tensor.

hstack

static hstack[track_grad: Bool = True](tensors: List[Tensor[dtype]], requires_grad: Optional[Bool] = None) -> Tensor[dtype]

Stack tensors horizontally (column-wise).

For 1D tensors: equivalent to concat(axis=0). For 2D+ tensors: equivalent to concat(axis=1).

Example: A: (3,), B: (3,) hstack([A, B]) → (6,)

A: (2, 3), B: (2, 5) hstack([A, B]) → (2, 8).

Args:

  • ​tensors (List): List of tensors to stack horizontally.
  • ​requires_grad (Optional): Whether to track gradients.

Returns:

Tensor: Horizontally stacked tensor.
